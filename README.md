# 🦁 Зоопарк теней

Детская игра-угадайка: на экране чёрная тень животного, нужно написать его название **на любом из 12 языков** (русский, английский, испанский, французский, немецкий, итальянский, португальский, украинский, польский, турецкий, китайский, японский).

Придумал игру ребёнок 7 лет. 🙂

## Правила
- Угадал — **+10 монеток** (в «золотом раунде», который выпадает случайно, — +20).
- Ошибся — **−2 или −3 монетки** (ниже нуля не бывает; повторное то же слово не штрафуется).
- Подсказки: 🔦 осветить кусочек тени (5🪙, до 3 раз), 🔊 услышать звук зверя (3🪙).
- 🎤 **Голосовой ввод**: кнопка-микрофон рядом с полем ответа — можно сказать название вслух (язык распознавания выбирается в списке под полем, по умолчанию русский). Работает через Web Speech API: Chrome/Edge/Safari; Chrome распознаёт через интернет; при первом нажатии браузер спросит разрешение на микрофон. Если микрофон услышал слово по-разному, игра честно проверяет все варианты — штрафа за «ослышался» почти не бывает.
- На заработанные монетки можно **покупать жителей в свой зоопарк** (цена растёт по мирам: 10–50🪙) — но только тех, кого уже угадывал.
- **7 миров**, открываются за всё заработанное (не потраченное!): Ферма — сразу, Джунгли — 50🪙, Океан — 100🪙, Лес — 150🪙, Север — 210🪙, Транспорт — 280🪙, Сказка — 360🪙. Всего **53 жителя**: звери, машины, ракета, робот, фея, динозавр…
- **Зоопарк — карта** с местами для жизни: Север ❄️, Лес 🌲, Полянка 🌾, Море 🌊, Дорога 🛣️. Купленный житель селится в родное место, но его можно **переселить** кнопками в его окошке.
- 🎙️ **Голосовушки**: нажми на жителя в зоопарке → «Записать голос» (до 10 секунд). Потом «▶️» — житель говорит твоим голосом, и у него **открывается рот в такт громкости**. Без записи говорит сам (синтез речи). Записи хранятся в IndexedDB браузера.
- Раз в день — 🎁 подарок +10 монеток.

## Как запустить
Просто открыть `index.html` двойным кликом — игра работает без интернета и без установки.
Или локальный сервер: `python3 -m http.server 4173` и открыть http://localhost:4173

Прогресс хранится в localStorage браузера. Кнопка «начать игру заново» — внизу вкладки «Миры».

## Как выложить в интернет (чтобы играли дети со всего мира)
Самый простой бесплатный способ — GitHub Pages:
1. `git init && git add . && git commit -m "Shadow Zoo"`
2. Создать репозиторий на GitHub, запушить.
3. Settings → Pages → Deploy from branch → `main` / root.
4. Через минуту игра будет по адресу `https://<логин>.github.io/<репозиторий>/`.

Альтернативы: itch.io (можно как «браузерная игра»), Netlify Drop (перетащить папку).

## Про «купить монетки за настоящие деньги»
Ребёнок предлагал добавить покупку монеток за реальные деньги. Сознательно **не реализовано**: встроенные покупки в детских играх регулируются законами (COPPA в США, аналогичные нормы в ЕС/РФ), требуют платёжной инфраструктуры, магазина приложений и родительского контроля. Вместо этого в игре есть ежедневный бесплатный бонус (+10). Если когда-нибудь захочется всерьёз — начинать со свежих гайдлайнов App Store / Google Play «Families».

## Если Safari «не спрашивает разрешения» на микрофон
Теперь при нажатии 🎤 игра сама вызывает системный запрос доступа к микрофону (`getUserMedia`), так что вопрос «Разрешить?» должен появиться. Если голос всё равно не работает, проверьте по порядку:
1. **Игра открыта как сайт**, а не двойным кликом по файлу: у `file://` Safari не даёт микрофон вовсе. Локально: `python3 -m http.server 4173` → http://localhost:4173 (игра сама подскажет это тостом).
2. **Включена Диктовка**: Системные настройки → Клавиатура → Диктовка (или включён Siri). Safari распознаёт речь через сервис Apple, и при выключенной Диктовке молча отдаёт ошибку `service-not-allowed` — без всякого диалога.
3. **Safari → Настройки → Веб-сайты → Микрофон**: для сайта игры должно стоять «Спрашивать» или «Разрешить» (если однажды нажали «Запретить», Safari больше не спрашивает).
4. Самый беспроблемный вариант — **Chrome**: там голос работает сразу (распознавание через интернет).

## Как менять игру
Всё в одном файле `index.html`:
- **Экономика** — константа `CONFIG` в начале скрипта (награды, штрафы, цены, пороги миров).
- **Жители** — массив `ANIMALS`: эмодзи (тень делается CSS-фильтром `brightness(0)`), мир `w`, звук-подсказка `snd`, названия `names` по языкам. Добавить жителя = добавить одну запись.
- **Миры** — массив `WORLDS` + пороги в `CONFIG.worldNeeds` + фон `.stage.wN` в CSS.
- **Места в зоопарке** — массив `ZONES` (id, название, декорации) и `HOME_ZONE` (какой мир куда селится по умолчанию).
- **Языки** — добавить ключ в `names` у жителей и флаг в `LANG_FLAGS`.
- **Голосовушки** — MediaRecorder → Blob в IndexedDB (`shadow-zoo-db/voices`, ключ — id жителя); воспроизведение через Web Audio с `AnalyserNode`, громкость которого открывает «рот» (див поверх эмодзи). Формат записи выбирается автоматически (`audio/mp4` в Safari, `audio/webm` в Chrome), поэтому запись и прослушивание работают в том браузере, где записывали.
